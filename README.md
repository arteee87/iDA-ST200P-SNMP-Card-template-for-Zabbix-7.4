# iDA-ST200P-SNMP-Card-template-for-Zabbix-7.4
For any UPS that can run the iDA-ST200P card. Card must be set to RFC-1628 mode.
These should be compatible:
EAST UPS EA900 G4 series
DEXP DEXP IEC Plus ONLINE series (tested)
SNR BASE (online) series (up to 3000VA models)
LANCHES L900Pro-S series

Import the template from the web ui as usual.
Triggers are pre-set for 230V 50Hz electrical. If you need to set other values - edit the triggers directly.
Change to macro values is planned.
Graphs and a basic panel is included.
